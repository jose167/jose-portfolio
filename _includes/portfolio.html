.net Framework

A web driven dynamic front end of a customer/stock management tool. This projec was built on the .net Framework using ASP.net (webforms), XML, JavaScript, HTML and CSS.

The website include the following requirements

  • Search facility for the website
  • Data entry screen to add a new customer
  • Add a validation routine to submitted forms for new stock item via data entry web form
  • Master/Detail screen of customers’ orders
  • Data Access Layer connected to the stock table; allow updates/edits/deletion
  • Web form that will add elements to an XML file, and a secondary linked web form that will allow a user to edit said XML file
  • Web form using an ADO.net connection to the ‘Suppliers’ table to display all relevant details

multimedia production

The initial project concept was to create a marketing campaign to re-brand the Cafe/Restaurant Third Space.

After researching competitors and identifying Third Spaces unique presence in the market place, I began the marketing campaign by first creating wire-frames to demonstrate the proposed layout of the website and storyboards to illustrate the structure of the promotional videos, whose primary focus would be to advertise the Third Space coffee product and raise awareness of events and gatherings held at the Restaurant.

A new logo was created as a vector graphic in illustrator which can be scaled to any size without losing detail. I also created the images for the website through the use of photography and image manipulation using Photoshop.

The website was designed using a combination of multimedia web technologies such as HTML5 and made responsive so it can be easily viewed on mobile devices.
